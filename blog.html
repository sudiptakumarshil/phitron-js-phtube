var, const, and let are both used to create variables but they have several implementations that are discussed below:
var keyword is function scoped, meaning it is global in a function.
For example:

function test(){
if(true){
var is_active = true;
}
console.log(is_active);
}
test();

let and const are similar to scops because they are block-scoped but const is different from let in one thing, const cannot be reinitialized but let can be initialized.
For let:

function test(){
if(true){
let is_active = true;
console.log(is_active);
}
console.log(is_active); // ERROR
} test();
let name ="Jhon";
name = "Doe";
console.log(name);
For const:

function test(){
if(true){
const is_active = true;
console.log(is_active);
}
console.log(is_active); // ERROR
}
test();
const name ="Jhon";
name = "Doe";
console.log(name);

Null and undefined are both data types in Javascript but they are different in their use cases.
Null: when a variable is defined but it does not have values.
Undefined: when a variable is not defined then it is called undefined.
